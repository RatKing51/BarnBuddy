# Farm Overview

Farm Overview gives a herd-level snapshot instead of a single animal record.

## What It Shows

Depending on account plan and available records, the overview can show:

- Total animals
- Active care flags
- Upcoming vaccinations
- Upcoming vet visits
- Herd status summaries
- Premium dashboard insights

## When To Use It

Use Farm Overview before chores, before a vet visit, or when checking whether a herd has upcoming work.
