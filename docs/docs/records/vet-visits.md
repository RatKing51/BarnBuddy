# Vet Visits

Vet visits record professional care, treatments, medications, and follow-up needs.

## Add a Vet Visit

1. Select an animal.
2. Open **Vet Visits**.
3. Select **Add**.
4. Enter the visit date, vet name, reason, and treatment.
5. Add medications, cost, follow-up date, and notes when needed.

## Follow-Ups

Follow-up dates can appear in upcoming care summaries until they are completed.

## Costs

Vet visit costs are included in Premium finance summaries when available.
