# Set Up Your First Herd

Herds keep animals grouped by farm, project, species, pasture, barn, class, or any structure that fits your workflow.

## Create a Herd

1. Open the dashboard.
2. Select **Settings**.
3. Open **Herd settings**.
4. Select **New Herd**.
5. Enter a herd name.
6. Add a description if helpful.
7. Save the herd.
8. Return to the dashboard.

## Tips

- Use clear names such as `Show Cattle`, `Goat Project`, `North Pasture`, or `FFA Market Hogs`.
- A deleted herd moves its animals to **Unassigned** instead of deleting the animals.
- Use **Unassigned** for animals that are between groups or not ready for a permanent herd.
