# Animal Records

Each animal has a profile that collects basic identity, health, vet, and Premium records.

## General Data

Use the **General Data** tab for:

- Name
- Tag number
- Species
- Sex
- Birth date and calculated age
- Weight
- Herd placement
- Behavior notes
- Comments
- Active or deceased status
- Profile photo

## Profile Photos

Upload a clear image that helps identify the animal. You can replace or remove the image later.

## Delete an Animal

Animal deletion is permanent. BarnBuddy asks for confirmation before deleting the animal and shows a toast when the delete succeeds.

Before deleting, consider whether marking the animal deceased is better for record history.
