# Inventory Tracking

Inventory is a Premium herd-level tool for tracking supplies used around the farm.

## Inventory Fields

- Item name and category
- Current quantity and unit
- Reorder level
- Cost per unit
- Supplier
- Expiration date
- Notes

## Stock Visibility

BarnBuddy flags items when the current quantity reaches the saved reorder level. It also highlights items that are expired or expire within 30 days.

## Open Inventory

Choose a herd, then open **Inventory** from the dashboard. Inventory saved under **Unassigned** is kept separate from named herds.
