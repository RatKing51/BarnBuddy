# Reproduction

Reproduction records help track breeding pairs, pregnancy checks, due dates, birth outcomes, and offspring.

## Breeding Records

A reproduction record can include:

- Dam
- Sire
- Breeding date
- Due date
- Outcome
- Breeding method
- Pregnancy check date
- Pregnancy status
- Birth date
- Offspring count
- Birth outcome
- Notes

## Parentage

BarnBuddy can store dam and sire relationships for an animal. This helps preserve lineage and connect birth history to the correct parents.

## Offspring

When adding offspring from a breeding record, BarnBuddy can create a new animal and link it back to the dam and sire.
