# Herd Finances

Herd Finances combines financial activity across the selected herd into one Premium reporting screen.

## Included Activity

- Animal-level income and expenses
- Feed costs
- Vet visit costs
- Net, income, and expense totals
- Expense category breakdown
- Monthly cash flow
- Searchable ledger rows

Use the filters to review all activity, income only, or expenses only. Records still belong to their original animal, feed, or vet entry.

## Export

Select the finance export action to prepare a printable PDF report with summary totals, charts, and detailed ledger tables.
