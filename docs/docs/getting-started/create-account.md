# Create an Account

BarnBuddy accounts are used to keep each farm's records private and tied to the correct user.

## Steps

1. Go to [barnbuddy.pro](https://barnbuddy.pro).
2. Select **Sign Up**.
3. Enter the requested account information.
4. Check your email for the verification code.
5. Enter the code to verify your account.
6. After verification, open the dashboard.

## After Signup

New accounts can start with the free plan. Free users can manage animal profiles, herds, health records, vet visits, and basic dashboard care status.

Premium features can be reviewed from the Pricing page or Account Settings.
