# FAQ

## Can I use BarnBuddy for one animal?

Yes. BarnBuddy works for a single project animal or a small herd.

## Can I organize animals by species?

Yes. Create herds for each species or use species fields inside each animal profile.

## Should I delete an animal or mark it deceased?

Use deceased status when you want to keep the animal's history. Delete only when you no longer need the record.

## Are Premium tools required?

No. Core animal, herd, health, and vet records are available on the free plan.

## Where do I get help?

Use the Contact page on the main BarnBuddy website and include your account email plus what you were trying to do.
