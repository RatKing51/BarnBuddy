# Add Your First Animal

Animals are the center of BarnBuddy records.

## Add an Animal

1. Open the dashboard.
2. Choose the herd where the animal should live.
3. Select **Add**.
4. Open the new animal.
5. Fill in the general data fields.

## Recommended Fields

- Name or tag number
- Species
- Sex
- Birth date
- Weight
- Herd
- Notes
- Profile photo

## Animal Status

Animals can be marked active or deceased. Deceased animals stay in the record system so their history can still be reviewed.
