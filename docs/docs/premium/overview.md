# Premium Overview

Premium expands BarnBuddy beyond basic records.

## Premium Includes

- Advanced dashboard analytics
- PDF and scheduled exports
- Automatic reminders
- Breeding, reproduction, and birth planning tools
- Finance records
- Feed planning tools
- Inventory tracking with low-stock and expiration visibility
- Shared farm account access when available

## Free Plan Includes

- Animal profiles
- Herd organization
- Health records
- Vet visit history
- Basic dashboard care status
- Mobile-friendly record keeping

## Upgrade

Open the Pricing page or Account Settings to review Premium.
