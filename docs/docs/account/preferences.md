# Preferences

Account preferences change how BarnBuddy feels and what information is emphasized.

## Available Preferences

- Dashboard density
- App theme
- Primary animal label
- Care window
- Product update emails
- Automatic reminders for Premium users

## Primary Animal Label

Choose whether the dashboard should emphasize animal name or tag number first.

## Dashboard Density

Comfortable density gives more spacing. Compact density fits more information on screen.
