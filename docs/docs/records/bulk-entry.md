# Bulk Entry

Bulk Entry creates the same vaccination or health event for several active animals at once.

## Add Records to Several Animals

1. Choose a herd.
2. Open **Bulk Entry**.
3. Search for animals or use **Select visible**.
4. Choose **Vaccination** or **Health event**.
5. Complete the record fields.
6. Review the number of records that will be created.
7. Select **Add to animals**.

Deceased animals are excluded from the selection list.

## Inventory Use

Premium users can connect an eligible inventory item and enter the amount used per animal. BarnBuddy multiplies that amount by the selected animal count, verifies available stock, and subtracts the total after the records are created.

Each selected animal receives its own record, so the entries can be edited separately later.
