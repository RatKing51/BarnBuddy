# Farm Overview

Farm Overview gives a herd-level snapshot instead of a single animal record.

![Farm Overview with care status and priority queue](/img/screenshots/farm-overview.png)

## What It Shows

- Total animals
- Stable rate and deceased count
- Vaccination and vet-care counts
- Care status split between overdue, due soon, stable, and deceased animals
- A priority queue linking directly to animals that need attention
- Species mix, age profile, and weight snapshot
- A filterable animal roster with status, species, tag, age, and weight

The **due soon** window comes from Account Settings and can be set to 7, 14, or 30 days.

## When To Use It

Use Farm Overview before chores, before a vet visit, or when checking whether a herd has upcoming work.
