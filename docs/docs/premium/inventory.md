# Inventory Tracking

Inventory is a Premium herd-level tool for tracking supplies used around the farm.

## Inventory Fields

- Item name and category
- Current quantity and unit
- Reorder level
- Cost per unit
- Supplier
- Expiration date
- Notes

## Stock Visibility

BarnBuddy flags items when the current quantity reaches the saved reorder level. It also highlights items that are expired or expire within 30 days.

## Connect Inventory to Care Records

An inventory item can be enabled for **Vaccinations**, **Health treatments**, or both. Once enabled, it appears as a stock option when creating those records.

Enter the amount used per animal. BarnBuddy subtracts that amount when the record is created. Bulk entry multiplies the amount per animal by the number of selected animals and checks that enough stock is available before creating any records.

## Open Inventory

Choose a herd, then open **Inventory** from the dashboard. Inventory saved under **Unassigned** is kept separate from named herds.
