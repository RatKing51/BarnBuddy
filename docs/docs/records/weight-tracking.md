# Weight Tracking

The **Weight** tab keeps a dated weigh-in history for each animal and turns those records into a trend chart.

![Example animal weight trend chart](/img/screenshots/weight-trend.png)

## Add a Weigh-In

1. Select an animal.
2. Open **Weight**.
3. Enter the date, weight, unit, and optional notes.
4. Select **Add Weight**.

BarnBuddy supports pounds and kilograms.

## What the Screen Shows

- Current weight
- Change since the previous weigh-in
- Average recorded weight
- Total change across the history
- A visual weight trend
- Editable weigh-in history

Existing rows save when you leave an edited field. Deleting a weigh-in permanently removes that point from the history and chart.
