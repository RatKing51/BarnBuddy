# Feed Records

Feed records help track herd-level feed purchases and planning.

## Feed Fields

- Record date
- Feed type
- Amount
- Unit
- Cost
- Next purchase date
- Notes

## Herd Feed

Open **Herd Feed** from the dashboard to manage feed records for the selected herd.

## Feed Reminders

If automatic reminders are enabled, feed records with upcoming next purchase dates can appear in reminder previews.
